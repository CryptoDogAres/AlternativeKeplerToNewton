# Proofread Checklist

Use this checklist for each proofreading request.

## Language

- Correct grammar and punctuation.
- Improve sentence clarity without changing technical meaning.
- Keep style consistent with surrounding sections.

## Facts

- Check chronology, names, proposition numbering, and direction of implications.
- Flag claims that lack support or appear to overreach the cited source.

## References

- Verify citation keys exist in bibliography files.
- Check whether each key supports the sentence where it appears.
- Flag missing citations for factual claims.

## Suggested New References

- Start from `refs/summaries/INDEX.md` unique one-hop list.
- Prefer references that directly strengthen unsupported or weakly supported claims.
- Provide brief rationale for each suggestion.
